Golang Introduction

Lets go through the following Hello World example. 


package main

import (
	"fmt"
)

func main() {
	fmt.Println("Hello Wold")
}
Program execution begins by initializing the main package and then invoking the function main. When that function invocation returns, the program exits. It does not wait for other (non-main) goroutines to complete.
  • To run the program, you can copy this code to helloWorld.go, and later you can use go run helloWorld.go.
  • import, this is used to import different packages. Example fmt.
