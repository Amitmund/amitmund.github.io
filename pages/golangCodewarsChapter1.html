    return x + y
}

Testing code. 

package main

import "testing"

func TestSum(t *testing.T) {
    total := Sum(5, 5)
    if total != 10 {
       t.Errorf("Sum was incorrect, got: %d, want: %d.", total, 10)
    }
}
Test case quick note

The first and only parameter needs to be t *testing.T

(usually the method under test i.e. TestValidateClient)

Calls t.Error or t.Fail to indicate a failure (I called t.Errorf to provide more details)

t.Log can be used to provide non-failing debug information

Must be saved in a file named something_test.go such as: addition_test.go

it need to start with Test followed by word starting with a capital letter.

Question

In this simple assignment you are given a number and have to make it negative. But maybe the number is already negative? 



Example:

MakeNegative(1)    # return -1
MakeNegative(-5)   # return -5
MakeNegative(0)    # return 0

Notes:

The number can be negative already, in which case no change is required.
Zero (0) is not checked for any specific sign. Negative zeros make no mathematical sense.
Answer 

package kata

func MakeNegative(x int) int {
  if x > 0 {
    return -x
  }
  return x
}
Another Answer 

package kata

import "math"

func MakeNegative(x int) int {
  return -int(math.Abs(float64(x)))
}

Question

Write a program that finds the summation of every number from 1 to num. The number will always be a positive integer greater than 0. 



Example:

summation(2) -> 3
1 + 2

summation(8) -> 36
1 + 2 + 3 + 4 + 5 + 6 + 7 + 8
Answer 

package kata

func Summation(n int) int {
  return n * (n+1) / 2
  }

Question

Clock shows 'h' hours, 'm' minutes and 's' seconds after midnight. Your task is to make 'Past' function which returns time converted to milliseconds. 



Example:

past(0, 1, 1) == 61000
Answer 

package kata

func Past(h, m, s int) int {
  return ((h * 3600) + (m * 60) + s ) * 1000
}
Another Answer 

package kata

    return (h*60*60+m*60+s)*1000
