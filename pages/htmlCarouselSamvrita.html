Quick note

				- Don’t autoplay
				- https://webflow.com/blog/carousel-slider-design-best-practices
				- Card style slider

				- https://mdbootstrap.com/docs/jquery/javascript/carousel/
				- The .active class needs to be added to one of the slides otherwise the .carousel will not be visible
				- Also be sure to set a unique id on the .carousel for optional controls, especially if you’re using multiple carousels on a single page
				- class="carousel slide carousel-fade"

				- Add captions to your slides easily with the .carousel-caption element within any .carousel-item.

				- Add .carousel-fade to your carousel to animate slides with a fade transition instead of a slide.
				- class="carousel slide carousel-fade"

				- https://getbootstrap.com/docs/4.0/components/carousel/
				
				- Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-interval="".
				- interval	number	5000.	The amount of time to delay between automatically cycling an item. If false, carousel will not automatically cycle.
				- keyboard	boolean	true.	Whether the carousel should react to keyboard events.
				- 
				$('.carousel').carousel({
					interval: 2000
				})
