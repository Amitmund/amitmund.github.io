Your keys ID
On the above command list-keys or list-secret-keys, you will get a hexadecimal key ID and this ID you can use, when you have to perform the command related to that keys. But note that, You can also use the Name or email id too.
In the following quick command I might be using the either ID or email-id as per the example.
To create your key pair

gpg --full-generate-key
Or you might like to use just the following command 

gpg --gen-key
Exporting your Public Key

gpg --output public.pgp --armor --export username@email
Exporting your Private Key

gpg --output private.pgp --armor --export-secret-key username@email
Note
Note, you can either use the given email id or haxadecimal id.
Also, Note, you can use -a in place of --armor.
Important Information
Also, note you might like to set a strong password while creating your key-pair, and I believe for personal use, I wpould like to keep my key not to get expire for ever.

What we can do, by exporting these keys. Keep it safe the privatekey and with the publickey, we can upload the same to a gpg publickey server, so that it can be used by other, if they want, you should be able to open that file and no one else, because, the original file is Signed by your publickey and you are the only one having the related privatekey.

Sharing your Public key

Now that we know, how to export our public key and private key, lets share out public key so that it can be used by other to encrypt the file or message, so that only you can able decrypt the same.

You can use some of the gpg key server, example keys.gnupg.net to upload your public keys, which can be fetch later by other as per the following example.

Fetching public key


gpg --keyserver keys.gnupg.net --recv-keys 6F4D1BEC8EF93D94
gpg: key 6F4D1BEC8EF93D94: public key "AmitKumarMund < email id >" imported
gpg: Total number processed: 1
gpg:               imported: 1

Once the key is fatched, you can check the same, using gpg --list-keys, which can be used later to encrypt the file, that can be shared to the owner of that public key.
